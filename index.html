Lottie Vectors is a program that runs under MatLab(r)
and lets you visualize vectors of different types in a variety of ways!
Versions below 2.2 take 2D vector data, versions of the program 2.2 and above (as released) use 3 dimensional vectors.
Both programs let you visualize / rotate / pan in 3D on screen. Below are links relating to Lottie Vectors for MatLab(r);
